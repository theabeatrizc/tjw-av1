Este projeto foi criado por Ana Beatriz Bezerra de Abreu Carneiro para a disciplina de tópicos para java web de 2021.2

Neste repositório está o projeto para a AV1 
Tal projeto tem como objetivo utilizar o modelo MVC para a criação de um repositório de empregados de uma empresa contendo ID, nome e sobrenome
para executar o projeto basta rodá-lo num servidor tomcat, acessando pela url o arquivo "employees.jsp", ou seja, a url de localhost deve finalizar com "/employees"
Para acrescentar e deletar registros, deve-se olhar para o "EmployeeService"
